[CALL]    - This text
[LSK]     - Open file
[RSK]     - Return/Exit
[0]       - Change font (3x5 or 8x8 or 8x16)
[Shift]   - Faster!
[Enter]   - Change view mode
[*]       - Top of the file
[#]       - End of the file
[5]       - File statistics
[7]       - Go to...
[8]       - Find...
[1]-[4]   - Select grid
[6]       - PageUp
[9]       - PageDown
